# digital-gate-pass
A secure e-Gate Pass Management System built with Node.js that allows users to register using Aadhaar-based identity verification and OTP authentication, apply for gate passes, and supports multi-level approval with role-based access.

Tech Stack
  Backend: Node.js, Express.js
  Database: MongoDB
  Auth: OTP + Password (JWT)
  Verification: Aadhaar (mock / API-based)
  Deployment: Docker, PM2

  QR Code Gate Pass
  Notifications: Firebase Cloud Messaging (FCM)
